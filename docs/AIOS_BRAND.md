# AIOS — Brand Identity

> The brand system. Visual direction confirmed (Cosmic Ledger). This codifies it
> so every surface — README, showcase, community, competition — looks like one thing.

## Name & one-liner

- **Name:** AIOS  (spoken "A-I-O-S"). Expansion when needed: *AI Operating System*.
- **One-liner:** **An AI agent that learns from every run.**
- **Tagline (poetic):** *Every run becomes a star.*
- **Category line:** the local-first operating layer for AI agents.

## Positioning

AIOS is not another agent or framework — it is the **operating layer around** any
agent (Claude/Codex/Gemini/local), giving it the two things a frozen model lacks:
**memory that compounds** and **reliability that's verifiable**. Symbiosis, not
competition. One system per person; it gets measurably smarter over time.

Anti-positioning: not a chatbot, not a wrapper SDK, not a cloud SaaS. Local-first,
sovereign, provable.

## Voice

Precise, quietly confident, evidence-first. Cosmic but not mystical — awe grounded
in receipts (Merkle roots, 1,512 sessions, benchmarks). Never hype; show the proof.
Korean-first with the founder; English for the world. Code/identifiers in English.

## Visual system — "Cosmic Ledger"

- **Mood:** deep space · living memory · a galaxy of agent runs · luminous network.
- **Palette:** bg `#05070f` / `#070a14`; cyan `#5eead4`; violet `#a78bfa`;
  amber accent `#ffd166`; ink `#eef1fa`; muted `#8b94b2`.
- **Type:** Space Grotesk (display/headings) + JetBrains Mono (code, data, hashes).
- **Core motif:** a single luminous **node** from which constellation lines radiate
  — *one memory becoming a galaxy*. This is the logo seed and the recurring texture.
- **Logo / wordmark:** "AIOS" with a glowing node-dot for the brand mark; gradient
  cyan→violet on near-black; works at 32px (favicon) and as a 512px app icon.
- **Hero:** WebGL nebula + animated memory-graph node network (live in
  `apps/showcase/index.html`).

Asset generation prompts (hero loop, app icon, OG card, 5-OS constellation, nebula
plate, pulse motif) are the vibe-focused prompts already drafted — feed to
Firefly/Sora/Runway. Keep every asset on the palette + motif above.

## Proof points (use these, not adjectives)

- 1,512+ sessions in the live public AkashicRecord, Merkle-verifiable.
- Open prediction API, no auth.
- 5 OS organs (MyWorld/HiveMind/MemoryOS/CapabilityOS/GenesisOS) under one kernel.
- 4 research-backed reliability pillars in the turn-loop (arXiv 2509.09677 / 2604.11978).

## Presence marker — "AIOS is here"

So you can *see* AIOS is running, everywhere:

- **Sigil:** `✦` (U+2726, four-pointed star) — *every run becomes a star*. The one
  glyph. Also the logo seed and the showcase node motif. One symbol, all surfaces.
- **CLI marker:** `✦ aios` — the output prefix (replaces ad-hoc `[aios]`/`[harness]`/
  `[cap-scan]`). e.g. `✦ aios onboard — …`, `✦ aios  horizon=short → qwen3:8b`.
- **Presence badge:** `✦ AIOS` for banners; `✦ AIOS active · every run becomes a star`
  as the session-active signal.
- **Single source:** `scripts/aios_sigil.py` (`SIGIL`, `mark()`, `badge()`,
  `active_line()`) — cyan on a TTY, plain otherwise; never hard-code the glyph.
- **Provenance:** AIOS-touched artifacts carry it (commits already use the
  Co-Authored-By trailer; receipts/outputs can stamp `✦`).

Rule: if AIOS did it, the `✦` shows. Live on: `aios --help`, `aios onboard`,
`aios do`/harness. Rolling out to serve / scanner / all remaining `[…]` prefixes.
