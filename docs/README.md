# MyWorld AIOS Docs

This folder is the shared reading surface for agents working across Hive Mind,
MemoryOS, and CapabilityOS.

Read order:

1. `AIOS_NORTHSTAR.md` — final system shape and role boundaries.
2. `AIOS_DEFINITION.md` — strict definition and anti-cheat invariants.
3. `AIOS_AGENT_PROTOCOL.md` — how agents write durable cross-repo records.
4. `AIOS_SMART_CONTRACT.md` — task contract between the three OS layers.
5. `AIOS_WORK_DISPATCH.md` — how myworld assigns work to lower repos.
6. `AIOS_BUILD_METHOD.md` — how Codex searches, directs, improves, and uses
   the OS layers while building AIOS.
7. `AIOS_AGENT_LEDGER.md` — append-only human-readable progress log.
8. `agents/*.md` — role-specific instructions for each OS agent.

These files are coordination documents. Implementation details still live inside
each repo.
