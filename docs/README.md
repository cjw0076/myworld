# MyWorld AIOS Docs

This folder is the shared reading surface for agents working across Hive Mind,
MemoryOS, and CapabilityOS.

Read order:

1. `AIOS_NORTHSTAR.md` — final system shape and role boundaries.
2. `AIOS_AGENT_PROTOCOL.md` — how agents write durable cross-repo records.
3. `AIOS_SMART_CONTRACT.md` — task contract between the three OS layers.
4. `AIOS_AGENT_LEDGER.md` — append-only human-readable progress log.
5. `agents/*.md` — role-specific instructions for each OS agent.

These files are coordination documents. Implementation details still live inside
each repo.
