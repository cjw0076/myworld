# AIOS Swarm North Star

The swarm target is **Sovereign Swarm AIOS**.

The phrase matters. This is not a shared brain, a central AI government, or a
global canonical memory. Each AIOS instance remains sovereign. Swarm behavior
emerges only from signed, redacted, reviewable projections that other local
instances may use as evidence.

## Core Framing

AIOS instances do not absorb each other's operating systems.

They may absorb only projections:

- MemoryOS: peer-signed memory draft projections, never raw accepted memory.
- CapabilityOS: capability observations with evidence refs, never tool
  authority.
- Hive Mind: sanitized run receipt projections, never remote execution rights.
- MyWorld: contract/result/ledger projections, never another control plane.

Each receiving AIOS maintains only a local posterior belief. Two well-behaved
instances may disagree. That is not a bug; it is the mechanism that preserves
sovereignty and diversity.

## First Layer

The first build layer is not identity and not sync. It is privacy projection:

```text
local record -> privacy validator -> redacted projection -> future share path
```

Until the validator exists and passes, there is no peer identity, no share repo,
no git fetch, no remote import, no MemoryOS acceptance, and no provider
execution.

## WWW Analogy

The web succeeded because independent sites could link without a central
authority. Sovereign Swarm AIOS should keep that openness, but avoid the web's
worst failure modes:

- no default public exposure
- no automatic trust
- no global ranking as truth
- no central super-peer authority
- no content farm equivalent for observations

## Success Condition

Sovereign Swarm is working when multiple independent AIOS instances can provide
evidence that improves a local decision without forcing a shared truth or
violating privacy. The output is better local judgment, not consensus.

## Non-Goals

- No raw memory federation.
- No shared root authority.
- No remote execution by trust alone.
- No universal canonical truth.
- No peer data import without review.
- No swarm dashboard before privacy, identity, manifest, and trust layers.
